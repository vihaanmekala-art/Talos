📥 Installation
Prerequisites

Python 3.10+ (for best performance)

A working Streamlit environment

Steps to Install

Download the Talos package (after purchase) and extract the contents.

Open a terminal (command line) in the directory where the project is located.

You should also download ollama and Gemma 3-1B quantized.

Install required dependencies:

pip install -r requirements.txt


Run the app:

streamlit run app.py

⚙️ Usage
1. Launching the App

To launch Talos, simply run:

streamlit run app.py


This will open the app in your web browser, where you can interact with all the features.

2. Navigating the Dashboard

Once the app is launched, you'll be able to:

Select from multiple tools like Stock Analysis, AI Assistant, System Monitoring, and more.

Input stock tickers for analysis, track live system performance, and set reminders.

Analyze financial data using built-in charting tools.

3. Interacting with the AI

Ask questions in natural language, such as:


"What does the MACD indicator say about Tesla's price movement?"

"Explain the RSI for this stock."

The AI assistant will provide insights based on your queries.
